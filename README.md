## Projects Overview

### Arithmetic Formatter
A script that receives a list of strings containing arithmetic problems and returns them formatted vertically and side-by-side for clearer readability.

### Bisection Method
An implementation of the bisection method numerical technique, used to approximate the roots of a continuous function.

### Convert Text to snake_case
A simple utility script that automates the conversion of camelCase or PascalCase strings into standard Python `snake_case`.

### Dijkstra Algorithm
A graph-based algorithm project focused on finding the shortest path between nodes in a weighted network.

### Expense Tracker
A lightweight command-line interface (CLI) application designed to log, categorize, and track daily expenditures.

### Hanoi Recursive Algorithm
A classic computer science problem solving the mathematical puzzle of the Tower of Hanoi using clean recursive logic.

### Merge Sort Algorithm
An implementation of the divide-and-conquer sorting method, breaking down collections to sort them efficiently.

### Password Generator
A tool that generates strong, randomized passwords based on custom constraints like uppercase letters, numbers, and special characters.

### Text Encryptor
A program capable of shifting, encoding, and decoding messages using fundamental cryptography concepts.

### Time Calculator
A function that adds a specific duration to a starting time, correctly accounting for AM/PM shifts and calculating the elapsed days.

### Verifying Card Number
A script utilizing the Luhn algorithm to check the mathematical validity of various credit and debit card formats.
